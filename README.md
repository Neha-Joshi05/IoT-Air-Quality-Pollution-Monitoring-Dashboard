## ✨ Features
- 🌫️ **PM2.5 & PM10** monitoring with EPA AQI calculation
- 🏭 **CO2 & Gas Index** from MQ135 sensor simulation
- 🌡️ **Temperature & Humidity** from DHT22 simulation
- 🎨 **Color-coded AQI Hero** — changes color with pollution level
- 📊 **Live Charts** — PM trend + AQI/CO2 trend
- ⚡ **Smart Alerts** — Hazard/Warning/Notice levels
- 📁 **CSV Logging** — full history saved
- 💻 **No Hardware Needed** — Python simulation

## 🚀 How to Run

```bash
git clone https://github.com/Neha-Joshi05/IoT-Air-Quality-Pollution-Monitoring-Dashboard.git
cd IoT-Air-Quality-Pollution-Monitoring-Dashboard
pip install -r requirements.txt

# Terminal only
python main.py

# Live dashboard
python dashboard/app.py
# Open http://localhost:5000
```
